# 🤖 Telegram Бот для Подсчета Калорий

Telegram бот для распознавания еды на фотографиях и подсчета калорий с использованием моделей YOLOv5/YOLOv8.

## 🌟 Возможности

- 📸 Распознавание еды на фотографиях
- 🔢 Подсчет калорий для обнаруженных блюд
- 📊 Информация о белках, жирах, углеводах и клетчатке
- 🎯 Поддержка более 100 различных блюд
- 🚀 Использование современных моделей компьютерного зрения

## 📋 Требования

- Python 3.8 или выше
- CUDA (опционально, для GPU ускорения)

## 🚀 Быстрый старт

### 1. Создание Telegram бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/botfather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Установка зависимостей

```bash
# Установите зависимости проекта
pip install -e .

# Или установите напрямую из requirements.txt
pip install -r requirements.txt
```

### 3. Настройка

Создайте файл `.env` в корне проекта:

```bash
cp .env.example .env
```

Откройте `.env` и добавьте ваш токен:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

### 4. Запуск бота

#### Вариант 1: С токеном из .env файла

```bash
python telegram_bot.py
```

#### Вариант 2: С токеном через аргумент командной строки

```bash
python telegram_bot.py --token YOUR_BOT_TOKEN
```

#### Вариант 3: С выбором модели

```bash
# Использование YOLOv5s (быстрая, легкая)
python telegram_bot.py --model yolov5s

# Использование YOLOv5l (более точная, медленнее)
python telegram_bot.py --model yolov5l

# Использование YOLOv8s (новая версия)
python telegram_bot.py --model yolov8s
```

## ⚙️ Параметры запуска

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| `--token` | Telegram Bot API токен | Из переменной окружения |
| `--model` | Модель для распознавания | yolov5s |

### Доступные модели

| Модель | Размер | Скорость | Точность | Рекомендация |
|--------|--------|----------|----------|--------------|
| yolov5s | ~28MB | Очень быстрая | Хорошая | Для начала |
| yolov5m | ~84MB | Быстрая | Очень хорошая | Баланс |
| yolov5l | ~189MB | Средняя | Отличная | Высокая точность |
| yolov5x | ~357MB | Медленная | Лучшая | Максимальная точность |
| yolov8s | ~22MB | Очень быстрая | Отличная | Новая версия |

## 🐳 Docker (опционально)

Создайте `Dockerfile`:

```dockerfile
FROM python:3.9-slim

WORKDIR /app

# Установка зависимостей
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Копирование кода
COPY . .

# Переменная окружения для токена
ENV TELEGRAM_BOT_TOKEN=""

CMD ["python", "telegram_bot.py"]
```

Запуск:

```bash
# Сборка образа
docker build -t calorie-bot .

# Запуск контейнера
docker run -e TELEGRAM_BOT_TOKEN=your_token calorie-bot
```

## 🐛 Решение проблем

### Бот не запускается

1. Проверьте правильность токена
2. Убедитесь, что все зависимости установлены
3. Проверьте логи на наличие ошибок

### Распознавание работает плохо

1. Используйте более тяжелую модель (yolov5l или yolov8s)
2. Фотографируйте еду сверху при хорошем освещении
3. Включите `tta=True` или `ensemble=True` для лучшей точности

### Бот работает медленно

1. Используйте более легкую модель (yolov5s)
2. Используйте GPU для ускорения
3. Отключите `tta` и `ensemble`

### Ошибка "Not enough memory"

1. Используйте более легкую модель
2. Увеличьте swap memory
3. Уменьшите размер входных изображений
